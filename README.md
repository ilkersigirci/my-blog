# Blog-Markdown

## Install

```bash
# Install bun
curl -fsSL https://bun.sh/install | bash

# Using bunx instead of using npx or pnpm dlx
bun create svelte@latest your_app

#Move to your app directory
cd your_app/
# Using bun pacakge manager
bun i

#start the dev server
bun dev

```

## Build

```bash
#build your app
bun run build

#preview your app
bun run preview
```
